# DIO | TEMA: Branding para Loja de Calçados
O estudo de interesse para o projeto em Inteligência Artificial Generativa, resume em como desenvolver um roteiro para trabalhar na criação de branding de uma loja de calçados. 
Elaborando estratégias, visão, plano de atividades e posicionamento como ponto chave.

# Questões
- Quais elementos-chave preciso definir para que minha marca seja percebida como única e relevante no mercado?
- Como estruturar uma estratégia de branding que alinhe propósito, posicionamento e diferenciação competitiva?
- Quais passos práticos devo seguir para traduzir o posicionamento da marca em uma narrativa clara e consistente para o público?

# Desafios na pesquisa
Montar um documento com relevância para um assunto complexo. Desenvolver o prompt para não receber respostas vagas ou genéricas. 

# Aprendizados
Conhecimento das etapas para construção de uma boa pesquisa com a inteligência artificial. Estrutura de prompt, aprofundamento e validação. Visão de negócios para direcionamento de prompt.

# Resumo
Definir cada etapa para construção do projeto alimenta o crescimento do Branding da empresa. Assim analisa melhor o mercado, conhece o publico, constrói arquitetura, melhora o engajamento e efetiva com os indicadores para dar continuidade no crescimento da companhia.

# Grossário dos principais conceitos
- Branding: processo estratégico de construção e gestão da marca.
- Proposta de Valor: promessa única que diferencia a marca no mercado.
- Posicionamento: percepção desejada da marca na mente do consumidor.
- Identidade Visual: conjunto de elementos gráficos (logo, cores, tipografia).
- Identidade Verbal: tom de voz, linguagem e narrativa da marca.
- Storytelling: técnica de contar histórias para criar conexão emocional.
- Arquitetura de Marca: estrutura organizacional das marcas e sub-marcas de um portfólio.
- Experiência do Cliente (CX): conjunto de interações que definem como o cliente percebe a marca.
- Governança de Marca: regras e processos que garantem consistência e longevidade da marca.
- KPIs (Key Performance Indicators): indicadores-chave para medir desempenho e impacto das ações de branding.
- NPS (Net Promoter Score): métrica de fidelização que avalia a probabilidade de recomendação da marca.
